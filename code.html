Just Code It!

Problem statement --   Write a program in any language to find the sum of the series 1 +11 + 111 + 1111 + ... n terms.
 Code :





Sample input :


1 + 11 + 111 + 1111 + 11111




Sample output :


The sum of the series is: 12345



  using namespace std;
  int main()
  {
  int n, i, sum = 0;
  int t = 1;
  cout << "\n\n Display the sum of the series 1 +11 + 111 + 1111 + .. n terms:\n";
  cout << "-------------------------------------------------------------------\n";
  cout << " Input number of terms: ";
  cin >> n;
  for (i = 1; i <= n; i++)
  {
  cout << t << " ";
  if (i < n)
  {
  cout << "+ ";
  }
  sum = sum + t;
  t = (t * 10) + 1;
  }
  cout << "\n The sum of the series is: " << sum << endl;
  }

Variables used : n, i, sum, t

complexity-- O(n) and not O(n2)
